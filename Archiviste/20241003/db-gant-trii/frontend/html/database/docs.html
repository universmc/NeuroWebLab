Configuration du Setup

Guides et tutoriels pour configurer votre environnement de développement

1. Prérequis

Avant de commencer, assurez-vous d'avoir les éléments suivants :

  • Un ordinateur avec système d'exploitation Unix ou Linux
  • Python installé
  • Git installé

2. Installation de Python et Git

Si vous n'avez pas encore installé Python et Git, suivez ces étapes :

  1. Téléchargez et installez Python depuis le site officiel : python.org
  2. Installez Git en suivant les instructions pour votre système d'exploitation : git-scm.com

3. Configuration de l'environnement

Configurez votre environnement de développement en suivant ces étapes :

  1. Clonez le dépôt Univers-MC depuis GitHub :
    git clone https://github.com/votre-utilisateur/univers-mc.git
  2. Naviguez vers le répertoire racine du projet :
    cd univers-mc
  3. Installez les dépendances Python :
    pip install -r requirements.txt
